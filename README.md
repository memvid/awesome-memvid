# Awesome Memvid [![Awesome](https://awesome.re/badge.svg)](https://awesome.re)

A curated list of awesome resources related to **memvid**.

## Resources

### Documentation

* [Memvid Documentation](https://docs.memvid.com/) – Official documentation for Memvid: single-file AI memory with hybrid search, entity extraction, and time-travel debugging

### Tutorials

* [memvid CLI - easy way to build RAG Pipelines](https://youtu.be/v0hDIPmFCgM?si=2j4vBj3C-ovHzSMa) – Demo on how to use memvid CLI for local search and building RAG pipelines

## Packages

* [memvid-cli](https://www.npmjs.com/package/memvid-cli) – Command-line interface for Memvid
* [@memvid/sdk](https://www.npmjs.com/package/@memvid/sdk) – Node.js SDK for Memvid
* [memvid-sdk](https://pypi.org/project/memvid-sdk/) – Python SDK for Memvid
* [commitreel](https://github.com/commitreel) – Time travel for checkpoints. Record a single MV2 tape, scrub a timeline, and run any moment on demand
* [memvid-mind](https://github.com/memvid-mind) – Give Claude Code photographic memory in ONE portable file. No database, no SQLite, no ChromaDB
* [adrflow](https://github.com/adrflow) – Capture architectural decisions while you code. An MCP server that records decisions and provides full context
* [canvas](https://github.com/canvas) – AI UI Kit powered by Memvid. Build AI-powered apps in minutes with React components and built-in memory management
